Jack Frost's Winter
Jack Frost's Winter is a 3 or more reels, 4-row video slot featuring Infinity Reels, increasing multipliers, and expanding Wild symbols.
The game is played with 10 base bet (fixed), bet level of 1 to 10 and bet size of 0.10 to 50.00.
The in-game currency is à¸¿.
The Bet Size is set using the "Bet Size" selector.
The Bet Level is set using the "Bet Level" selector.
The Bet Amount is set using the "Bet Amount" selector.
"Cash Wallet" displays the cash available for wager.
"Auto Spin" automatically plays the game for a selected number of rounds.
At every spin starting with 3 reels, any combinations of 5 or more symbols (excluding Free Spins symbol) from the leftmost reel will result in a winning combination.
On any spin, if the rightmost reel improves any winning combination, then an extra reel is added to the right.
On any spin, if the rightmost reel does not improve any winning combination and there is a Free Spins symbol on the rightmost reel, then an extra reel is added to the right.
The added extra reel is spun, and if it again improves any winning combination, then another extra reel is added and so on.
Wins are calculated once no extra reel is added.
For every winning combination, payout is made for each symbol according to the "Paytable".
A combination win in cash is equal to the value shown in the "Paytable" multiplied by bet size and bet level.
All wins shown in cash.
Wild symbol substitutes for all symbols except Free Spins symbol.
Wild symbol appears on all reels except reel 1.
Free Spins Feature is played at the same bet size and bet level as the spin that triggered the Free Spins Feature.
Multiplier
At the start of any main game spins, the win multiplier is x1.
During any main game spins, the win multiplier increases by 1 for every extra reel added.
Wins are then multiplied by the win multiplier and paid once no extra reel is added.
Expanding Wild Feature
Any Wild symbol that appears on the reels and is involved in any win will expand across entire reel.
Free Spins Feature
3 Free Spins symbols appearing anywhere will trigger the Free Spins Feature with 10 free spins. Each additional Free Spins symbol will trigger 2 more free spins.
At the start of the Free Spins Feature, the win multiplier is x1.
During any free spin, the win multiplier increases by 1 for every extra reel added.
1 free spin is awarded for each Heart symbol on the reels (if any).
Win multiplier is increased by 1 for each Star symbol on the reels (if any).
